## DSList - Game Management API
Bem-vindo ao repositório DSList, uma API para gerenciamento de listas de jogos. Este projeto foi desenvolvido utilizando o framework Spring Boot, com integração a um banco de dados relacional, seguindo boas práticas de desenvolvimento.

## Funcionalidades
A API permite:

- Listar todos os jogos disponíveis.
- Consultar detalhes de um jogo específico.
- Gerenciar listas de jogos:
- Listar todas as listas de jogos.
- Obter os jogos de uma lista específica.
- Reordenar jogos dentro de uma lista.
## Tecnologias Utilizadas
- Java 17
- Spring Boot 3
- Spring Data JPA
- Spring Web
- Spring DevTools
- Banco de Dados: MySQL
- Maven: Gerenciador de dependências
- JPA / Hibernate: Para mapeamento objeto-relacional
- Postman: Testes de API

## Endpoints da API
## Games
- GET /games: Lista todos os jogos com informações mínimas.
- GET /games/{id}: Retorna os detalhes de um jogo específico.
## Game Lists
- GET /lists: Lista todas as listas de jogos.
- GET /lists/{listId}/games: Lista os jogos pertencentes a uma lista específica.
- POST /lists/{listId}/replacement: Move jogos dentro de uma lista, alterando a ordem.

### WebConfig

```java
@Configuration
public class WebConfig {

	@Value("${cors.origins}")
	private String corsOrigins;
	
	@Bean
	public WebMvcConfigurer corsConfigurer() {
		return new WebMvcConfigurer() {
			@Override
			public void addCorsMappings(CorsRegistry registry) {
				registry.addMapping("/**").allowedMethods("*").allowedOrigins(corsOrigins);
			}
		};
	}
	
}
```

### GameRepository

```java
@Query(nativeQuery = true, value = """
		SELECT tb_game.id, tb_game.title, tb_game.game_year AS `year`, tb_game.img_url AS imgUrl,
		tb_game.short_description AS shortDescription, tb_belonging.position
		FROM tb_game
		INNER JOIN tb_belonging ON tb_game.id = tb_belonging.game_id
		WHERE tb_belonging.list_id = :listId
		ORDER BY tb_belonging.position
			""")
List<GameMinProjection> searchByList(Long listId);
```
